## Описание программы
Currency-Converter — это простое и удобное приложение для конвертации валют с графическим интерфейсом, написанное на Python с использованием библиотеки *tkinter*. Программа позволяет быстро переводить суммы между различными валютами, используя актуальные курсы, получаемые из онлайн-API. Все операции сохраняются в историю, которую можно просмотреть прямо в окне приложения.

## Возможности программы
- Конвертация между популярными валютами (*USD*, *EUR*, *GBP*, *JPY*, *CNY*, *RUB*).
- Получение актуальных курсов валют через интернет.
- Валидация введённых данных (сумма должна быть положительным числом).
- Сохранение истории операций в локальный файл.
- Отображение истории в виде таблицы.

## Инструкция по запуску
1. Убедитесь, что у вас установлен Python 3.
2. Установите необходимые библиотеки:
   ```bash
   pip install requests
   ```
3. Сохраните код в файл с расширением `.py` (например, `currency_converter.py`).
4. Запустите файл:
   ```bash
   python currency_converter.py
   ```
5. В открывшемся окне выберите валюты, введите сумму и нажмите «Конвертировать».

## Требования
- **Операционная система:** любая, поддерживающая Python 3 и *tkinter*.
- **Python:** версия 3.6 или выше.
- **Интернет-соединение:** для получения актуальных курсов валют.

## Примеры использования
1. Перевести 100 долларов в евро:
   - В поле «Из» выберите *USD*.
   - В поле «В» выберите *EUR*.
   - Введите сумму: `100`.
   - Нажмите «Конвертировать».
2. Перевести 5000 рублей в юани:
   - В поле «Из» выберите *RUB*.
   - В поле «В» выберите *CNY*.
   - Введите сумму: `5000`.
   - Нажмите «Конвертировать».

## Тестирование
Программа тестировалась на следующих сценариях:
- Корректная конвертация между всеми поддерживаемыми валютами.
- Обработка ошибок при вводе некорректной суммы (например, отрицательное число или текст).
- Проверка работы с недоступным интернетом (выводится сообщение об ошибке).
- Сохранение и отображение истории операций после перезапуска программы.
